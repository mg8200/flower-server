## 1.项目安装
```
npm install
```

```
cnpm install
```

### 2.导入数据库文件

数据库文件在mysql文件中`flowersbaking.sql`

### 3.修改数据库配置

设置mysql文件下的index.js中的相应配置即可

### 4.下载相应的静态资源

下载地址：链接:http://www.minggang.top:8200/down/5vBAhxilUenl 提取码:6aZmmN

### 5.服务器启动

```
node ./index.js
```

npm start
